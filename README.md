# NeoElite Fleet Management Dashboard

A full-stack Next.js application for managing line and dedicated vehicles, logging daily statuses, and tracking idle utilization.

## Features
- **Management Dashboard**: Secure portal for tracking fleet utilization.
- **Supervisor Portal**: Mobile-friendly application for bulk updating daily vehicle logs.
- **Admin Panel**: Full master database controls for vehicle configuration.

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Seed the database (optional if already seeded):
```bash
node init-db.js
```

3. Run the development server:
```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
